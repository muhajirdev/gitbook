Python Basic

Python merupakan salah satu bahasa pemrograman yang paling simple. Coba bandingkan saja dengan Java.

Java :

public class TestClass {
  public static void main(String[] args) {
    System.out.print("Hello World");
  }
}

Python :

print "Hello World"

Bagaimana cukup sederhana bukan? Yap, karena kesederhanaan inilah python menjadi salah satu bahasa pemograman paling populer di dunia, ya walaupun java juga cukup populer sih :D.

Hal lainnya yang menarik dari python adalah kita tidak perlu menyebutkan type variable yang akan kita gunakan contohnya, di java' misal kita ingin men-*declare* variablenamadengan type varible berupaString`, maka kita perlu menyebutkan typenya.

String nama = "Muhammad Muhajir";

atau lebih parahnya lagi 

String name = new String("Muhammad Muhajir");

Tapi di python, As simple as Hell.

nama = "Muhammad Muhajir"

Problem solved.

Another interesting point dari python : Tidak perlu semi colon (;)

Seperti pada contoh di atas, setiap statement di java selalu di akhir dengan semi colon (;)

Java :

System.out.print("Hai Kamuu");

Python:

print "Hallo"

Interesting point yang ke 3: RAPI

Di Python, code yang anda tulis wajib rapi. Kenapa wajib?? Di java "space" itu tidak berpengaruh. Misal :

Java :

public void setAge(newAge){
  age = newAge;
}

sama saja dengan 

public void setAge(newAge){age = newAge;}

Sedangkan di python, karena tidak menggunakan curly brace "{" dan "}" untuk menyatakan bagian statement , python menggantinya dengan penggunaan space.

Python :

def saySomething(something):
  print something
  print "bello"

Jadi di python, code di atas dan code di bawah adalah 1 hal yang berbeda. Code di atas menegaskan bahwa print "bello" masih bagian dari function saySomething, sedangkan pada code bawah, print "bello" telah berdiri sendiri dan bukan merupakan bagian dari saySomething.

def saySomething(something):
  print something
print "bello"
